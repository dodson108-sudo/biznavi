자주 묻는 질문

DART(금융감독원 전자공시시스템)는 상장사·대형 법인의 재무제표를 공시하는 정부 데이터베이스입니다. BizNavi는 109,030개 기업 목록을 연동하여 회사명만 입력하면 재무제표를 자동으로 불러옵니다. 비상장 중소기업은 공시 의무가 없어 DART에 재무데이터가 없을 수 있으며, 이 경우 직접 입력 모드로 진행하실 수 있습니다.
한국은행 ECOS(경제통계시스템)의 기업경영분석 데이터를 실시간으로 연동합니다. 분석 대상 기업의 업종코드를 기준으로 해당 업종의 산업평균 재무비율을 자동으로 가져와 비교합니다. 업종 데이터가 없는 경우 전체 제조업 평균을 기본값으로 사용합니다.
재무분석은 API 키 없이 완전히 사용 가능합니다. 경영전략 수립은 데모 모드로 샘플 분석 결과를 체험하실 수 있으며, 실제 맞춤형 AI 분석을 원하신다면 Anthropic Claude API 키를 등록하시면 됩니다.
모든 데이터는 사용자의 브라우저(localStorage)에만 저장됩니다. 외부 서버로 전송되거나 저장되지 않으므로 기업 재무 정보와 경영 전략 내용이 외부로 유출될 걱정 없이 사용하실 수 있습니다.
재무분석 리포트: 6대 재무비율 분석표, 업종 평균 비교, 종합 의견이 포함됩니다. 경영전략 보고서: SWOT 매트릭스, STP 분석, 4P 전략, 5대 역량 레이더 차트, KPI 대시보드, 실행 로드맵, 90일 플랜이 포함됩니다. 브라우저 인쇄 기능으로 PDF 저장이 가능합니다.
뿌리제조·식품제조·생활밀착서비스·전문유통도소매·외식·지식서비스IT·소규모건설·의료·교육·패션·미디어·물류운송·환경에너지·농림식품·수출중소기업·금융 등 16개 업종을 지원합니다. 각 업종마다 현장 상황에 맞는 BARS 5단계 앵커 진단 문항이 제공됩니다.
